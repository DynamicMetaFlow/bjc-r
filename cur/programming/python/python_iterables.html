Iterables in Python

In Snap!, we have two different for blocks. One of them iterates over a range of numbers, and the other one (for each item of) iterates over a list.

In Python, there is only one for construct and it behaves like for each item of in Snap!. for is more powerful than its Snap!em equivalently, since it can iterate over many different types of objects, including lists, strings, and more.

We use the term iterable to refer to any object that we can iterate over using the for keyword. For example, we can iterate over a range: 


for i in range(0,5):
    print(i)
We can iterate over a list: 

>>> names = ["Larry", "Curly", "Moe"]
>>> for item in names:
...	    print("Hello, " + item)
Hello Larry
Hello Curly
Hello Moe

And we can iterate over a string: 


>>> chant = "Go Bears!"
>>> for letter in chant:
...     print(letter)
